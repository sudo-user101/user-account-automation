# user-account-automation
User Provisioning Tool: A Python script and toolkit designed to automate user account management and provisioning within organizations, facilitating the generation of user email addresses, department names, and random passwords to streamline onboarding processes, enhance security, and improve efficiency in corporate environments.

# User Provisioning Tool

Automate user account management and provisioning within your organization effortlessly with the User Provisioning Tool. This Python script and toolkit simplify the process of onboarding new employees by generating user email addresses, department names, and random passwords.

## Features

- **User Email Generation:** Create unique email addresses for new users based on their name, surname, and department.
- **Department Name Management:** Easily assign department names to users, ensuring clarity and organization.
- **Random Password Generation:** Enhance security by automatically generating strong, random passwords for user accounts.
- **Streamlined Onboarding:** Speed up the onboarding process with automated user account setup, improving efficiency.
- **Customizable and Extendable:** Tailor the tool to fit your organization's specific requirements and easily extend its functionality.

## Usage

1. **Installation**: Clone this repository to your local machine and ensure you have Python installed.

2. **Configuration**: Modify the `user_provisioning_tool.py` script to match your organization's naming conventions, if necessary.

3. **Execution**: Run the script and follow the prompts to input user information, including name, surname, company, and department.

4. **Output**: The tool will generate a user email address and a random password for each user, streamlining the onboarding process.
